## 五毒神掌

- 5-10 min 自己写，写不出来看题解（要看国际版的 Top 3 回答） 
- 自己写
- 24 h
- 1 week
- 面试前

## 四步解题

- Clarification：明确题目的意思、边界、数据规模，可以和面试官沟通这些问题
- Possible solutions：
  - 提出可能的所有解法，并比较时间、空间复杂度
  - 然后用最优的方法解决问题
- Coding：代码简洁

- Test cases：写完后提供测试案例

## 面试技巧

- 和其他候选人，而不是和题目竞争
- 面试的过程是合作的关系，让对方感觉与你合作是很舒服的事情
- 开胃菜的题目需要快速解决掉，才会有后续



Tips：

- 保持 LeetCode 每日一题
- 多看别人的优秀代码，并记录总结
- 不要人肉递归，找出重复性
